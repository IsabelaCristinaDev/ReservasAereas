Sistema de Notificações de Voos — Padrão Observer
Este projeto implementa o Padrão de Projeto Observer aplicado a um sistema de notificações de voos, desenvolvido em Java com interface gráfica Swing

Padrão de Projeto: Observer
O Observer é um padrão comportamental que define uma dependência um-para-muitos entre objetos: quando um objeto muda de estado,
todos os seus dependentes são notificados e atualizados automaticamente.
Tecnologias Utilizadas

Java — linguagem principal
Java Swing — interface gráfica 
Padrão Observer  — arquitetura de notificações
